Task Tracker

Jakarta EE demo application

Основные сценарии

  • Пользователи
    • Просмотреть всех пользователей
  • Проекты
    • Просмотреть все проекты
  • Задачи
    • Просмотреть все задачи
    • Доступные статусы задач

Создание сущностей

Создание выполняется через REST API (POST-запросы). Для демонстрации можно использовать Postman или curl.

  • POST /api/users
  • POST /api/projects
  • POST /api/tasks

Описание сценариев

  1. Пользователь открывает стартовую страницу
  2. Просматривает список задач
  3. Создаёт новую задачу
  4. Изменяет статус задачи
  5. Удаляет задачу
