# Практическая работа №1

## Приложение с типовой архитектурой JakartaEE

**Geo-Dispatch** - сервер, который принимает пинги от трекеров машин, следит за геозонами-точками, продвигает наряды по
статусам и ведёт историю визитов. 

Умные фишки: адаптивные радиусы зон и подтверждение визитов по реальному “времени
пребывания”.
